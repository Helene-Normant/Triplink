### Installation

1. Clone the repo
 
 `git@github.com:Helene-Normant/Triplink.git`
 
3. Install packages

`docker exec -it php_triplink sh`
  
`composer install`

4. Run the project
  
`docker-compose build`

`docker-compose up -d`
